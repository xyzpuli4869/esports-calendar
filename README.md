# 电竞赛事日历 🎮

一个现代化的电竞赛事日历应用，支持查看 LOL、CS2、Valorant 的比赛信息。

## ✨ 功能特点

- 📅 日历视图查看比赛
- 🎯 游戏类型筛选
- 🏆 赛事筛选
- 🖼️ 战队Logo显示
- 📊 比赛比分显示
- 🌐 多语言直播源
- 🔒 API密钥本地保存
- 🇨🇳 中文赛事名称

## 🚀 在线演示

[https://esports-calendar-phi.vercel.app/](https://esports-calendar-phi.vercel.app/)

## 🛠️ 技术栈

- React
- Tailwind CSS
- PandaScore API
- Vercel Serverless Functions

## 📦 本地运行

1. 克隆仓库
\`\`\`bash
git clone https://github.com/xyzpuli4869/esports-calendar.git
cd esports-calendar
\`\`\`

2. 安装依赖
\`\`\`bash
npm install
\`\`\`

3. 启动开发服务器
\`\`\`bash
npm start
\`\`\`

4. 访问 http://localhost:3000

## 🔑 获取API密钥

1. 访问 [PandaScore](https://pandascore.co/)
2. 注册账号
3. 获取免费API密钥（1000次/月）

## 📝 部署

本项目使用 Vercel 部署：

\`\`\`bash
npm install -g vercel
vercel --prod
\`\`\`
